# Orga.com

SaaS de gestion de missions/plannings.
Stack: FastAPI + Postgres (backend), React + Vite + TypeScript (frontend).
Windows-first (PowerShell).

## Prerequis
- Windows 10/11, PowerShell 7+
- Python 3.10+ (vise 3.12), pip
- Node 20 LTS + pnpm ou npm
- Git
